## Architecture

```
address string
    │
    ▼
[ Nominatim ] ──► {lat, lon, display_name}
    │
    ├──► [ Overpass ]   ──► amenities[], buildings[], transit[], landuse[]
    ├──► [ BuildData ]  ──► permits[]  (filtered by 500m)
    ├──► [ 311 ]        ──► complaints[]  (Toronto only)
    ├──► [ Census ]     ──► demographics  (US only, optional)
    └──► [ OpenWeather ]──► airQuality  (optional)
    │
    ▼
[ score engine ]       ──► LivabilityScore (0-100, 5 components)
[ anomalies engine ]   ──► Anomaly[]   (z-score)
[ forecast engine ]    ──► Trend[]     (linear regression)
[ whatif engine ]      ──► ScenarioResult[] (toggleable)
    │
    ▼
UI: Map | ReportCard | AnomalyList | ForecastChart | WhatIfSimulator | ChatBox
```

### 1. `src/lib/types.ts`

Shared types consumed by every other file.

```typescript
// Coordinates
export type LatLon = { lat: number; lon: number };

// Geocoding
export interface GeocodeResult extends LatLon {
  displayName: string;
  bbox?: [number, number, number, number];
}

// OSM
export type AmenityKind =
  | 'restaurant' | 'cafe' | 'school' | 'grocery' | 'park'
  | 'bus_stop' | 'transit' | 'construction';

export interface Amenity {
  id: string;
  kind: AmenityKind;
  name?: string;
  lat: number;
  lon: number;
  tags: Record<string, string>;
}

export interface OverpassResponse {
  amenities: Amenity[];
  buildings: Amenity[];
  transit: Amenity[];
  landuse: Amenity[];
  rawCount: number;
}

// Permits & complaints
export interface Permit {
  id: string;
  address: string;
  lat: number;
  lon: number;
  description: string;
  issuedDate: string;
  structureType: string;
  constructionValue?: number;
  status: string;
}

export interface Complaint {
  id: string;
  type: string;
  date: string;
  lat: number;
  lon: number;
  status: string;
}

// Signals & scoring
export interface ScoreBreakdown {
  amenityDensity: number;
  transitScore: number;
  foodAccess: number;
  greenSpace: number;
  development: number;
}

export interface LivabilityScore {
  total: number;
  breakdown: ScoreBreakdown;
  cityAverage: number;
}

// Anomalies
export interface Signal {
  name: string;
  current: number;
  baseline: number;
  unit: string;
}

export interface Anomaly {
  signal: string;
  zscore: number;
  severity: 'warning' | 'critical';
  message: string;
}

// Forecast
export interface Trend {
  signal: string;
  current: number;
  slope: number;
  history: number[];
  forecast6m: number;
  forecast12m: number;
  forecast24m: number;
  r2: number;
  confidence: 'high' | 'medium' | 'low';
}

// What-if
export interface Scenario {
  id: string;
  name: string;
  emoji: string;
  description: string;
  impact: Partial<ScoreBreakdown>;
}

export interface ScenarioResult {
  scenarioId: string;
  before: number;
  after: number;
  delta: number;
  modifiedBreakdown: ScoreBreakdown;
}

// Full report
export interface NeighborhoodReport {
  address: string;
  coords: LatLon;
  fetchedAt: string;
  score: LivabilityScore;
  amenities: OverpassResponse;
  permits: Permit[];
  complaints: Complaint[];
  anomalies: Anomaly[];
  trends: Trend[];
  sources: {
    overpass: 'ok' | 'failed' | 'partial';
    builddata: 'ok' | 'failed' | 'skipped';
    complaints: 'ok' | 'failed' | 'skipped';
    census: 'ok' | 'failed' | 'skipped';
    weather: 'ok' | 'failed' | 'skipped';
  };
}

// Chat
export interface ChatRequest {
  question: string;
  report: NeighborhoodReport;
}

export interface ChatResponse {
  answer: string;
  modelUsed: string;
}
```

## Data Flow

```
GET /api/report?address=123+Queen+St+W+Toronto
   │
   ▼
[1] cache.get(key) → hit? return
   │
   ▼
[2] nominatim.geocode('123 Queen St W Toronto')
   │   → {lat: 43.647, lon: -79.393, displayName: '...'}
   │
   ▼
[3] Promise.allSettled([
       overpass.fetchOverpass({lat,lon}),
       builddata.fetchPermits({lat,lon}),
       complaints.fetchComplaints({lat,lon}),
       census.fetchCensus({lat,lon}),
       weather.fetchAirQuality({lat,lon})
     ])
   │
   ▼
[4] score.computeBreakdown(amenities, permits)
   │   → { amenityDensity: 78, transitScore: 65, ... }
   ▼
   score.computeTotal(breakdown) → { total: 72, ... }
   │
   ▼
[5] build 12-month signal series:
       permitsByMonth = group(permits, 'YYYY-MM')
       complaintsByMonth = group(complaints, 'YYYY-MM')
   │
   ▼
[6] anomalies.detectAnomalies([...signals...])
   │
   ▼
[7] forecast.forecastTrend(permitsByMonth, 'permits')
   forecast.forecastTrend(complaintsByMonth, 'complaints')
   │
   ▼
[8] cache.set(key, report, 5min)
   return NextResponse.json(report)
```

## API Call Order on User Submit

1. Browser → GET /api/report?address=... (one round trip)
2. Server → Nominatim (geocode)
3. Server → 5 parallel fetches via allSettled
4. Server → engines (pure functions, <10ms)
5. Server → NextResponse.json
6. Browser renders
7. On chat submit: POST /api/chat → OpenAI (or Ollama)
8. On what-if toggle: pure client-side, no network
